Introduction to BVE

1. What is BVE?

BVE (Brayden's Virtual Environment) is just like Python's IDLE, except it's not built-in, and you can have predefined libraries and what we like to call "BVE functions", which are just glorified pieces of code. These BVE functions are also predefined. You make them yourself and everytime you open BVE you can access any of your functions.

2. Python function Vs. BVE function

A "BVE function" refers to BVE's concept of a function and not necessarily standard Python function concepts. BVE's "function" is supposed to be nested Python functions and classes within an "origin" class. Basically, this "origin" class is where all your BVE function data is contained. For user-experience, please try to keep your origin class name the same as the name of the file containing the BVE function.

3. Can my BVE function access another BVE function?

BVE functions can access eachother, allowing for collaboration or additions to BVE functions from people who choose to use BVE. If standard BVE function layout is used, which is described in our "Python function Vs. BVE function" section,  BVE functions may be accessed just like they are in the environment, addressing them by the name of their origin class.

4. What are all these files for?

Let's break it down nice and simple. BVE.py is the virtual environment, it is BVE, that's all you need to know about it. There's another file, "libraries.txt", this contains all the predefined libraries! Add or remove whatever libraries you want in there, remember to separate them by putting them on their own line. Please don't put anything before or after them on the same line if possible. Last but not least, we have our folder called "functions", this contains a text file with each of our BVE functions inside of it. By now you've probably read all about them. If you need more instruction and guidance on these, there's a function called "exampleFunction.txt" you should check out! It can be found in the "functions" folder.

5. How do I access BVE functions? How do I use them?

You can use BVE functions just like they're a python class! That is, if they use the standard formatting for a BVE function. If you can't tell, I'd really like you to use a standardized format. It just makes everything so much easier.
